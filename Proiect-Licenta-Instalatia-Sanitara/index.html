Proiect Licenta - Instalatia Sanitara

Sistem experimental bazat pe ESP32 pentru monitorizarea si controlul a doua rezervoare, cu senzori de nivel, debitmetru, valva, pompa bidirectionala, afisaj OLED si integrare MQTT.

Ce face proiectul

Firmware-ul citeste nivelul din doua rezervoare, masoara debitul de apa, controleaza valva si pompa, afiseaza local starea pe OLED si comunica prin MQTT cu restul sistemului din licenta.

Proiectul poate rula in modul AUTO, in care actioneaza dupa pragurile configurate, sau in modul MANUAL, unde actuatorii sunt controlati prin mesaje MQTT.

Arhitectura pe scurt

Rezervoare

Doua recipiente, R1 si R2, fiecare cu senzor ultrasonic si plutitor.

Linia 1

R1 -> debitmetru -> valva -> R2

Linia 2

R1 <-> pompa <-> R2

Comunicatie

Control si status prin MQTT pe topicurile dedicate proiectului.

Componente principale

  • ESP32 Development Board cu OLED 0.96 inch
  • 2 x JSN-SR04T pentru nivel
  • 2 x plutitor
  • 1 x debitmetru
  • 1 x valva electromagnetica 12V NC
  • 1 x pompa bidirectionala
  • driver pentru valva si driver pentru pompa

Schema de cablare si recomandarile pentru alimentare sunt disponibile in hardware.html. Schema vizuala completa este disponibila in schema-instalatie.html.

MQTT pe scurt

Comenzi primite pe LICENTA/SANITAR/COMANDA

Status publicat pe LICENTA/SANITAR/STATUS

Firmware-ul publica starea scurta ABC, nivelurile NIVEL-1-XYZ, NIVEL-2-XYZ si debitul real DEBIT-XYZ.

Schema logica pentru topicuri, mesaje si tranzitiile dintre AUTO si MANUAL este disponibila in mqtt.html.

Configurare recomandata

  • WiFi SSID: Licenta-RPi-AP
  • WiFi password: licenta-rpi
  • MQTT host: 192.168.4.1
  • MQTT port: 1883
  • Client ID: licenta-sanitar-esp32
