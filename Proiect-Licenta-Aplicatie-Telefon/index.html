Proiect Licenta - Aplicatie Android Control Casa

Aplicatie mobila pentru controlul si monitorizarea subsistemelor Acces, Iluminat si Instalatie Sanitara din proiectul de licenta.

Prezentare Arhitectura Functionalitati Comunicatie Configurare Documentatie

Prezentare

Aplicatia Android este interfata de comanda pentru casa inteligenta din proiectul de licenta. Ea trimite comenzi catre brokerul MQTT local, primeste statusurile publicate de subsisteme si afiseaza informatii utile pentru operator.

Aplicatia este construita in Kotlin cu Jetpack Compose si este gandita pentru rulare pe telefon Android conectat la reteaua locala a Raspberry Pi.

Arhitectura generala

Android

Aplicatia mobila

Afiseaza paginile de control, trimite comenzi MQTT si actualizeaza interfata pe baza statusurilor primite.

MQTT

Broker local

Brokerul ruleaza in reteaua Raspberry Pi si transporta comenzile si statusurile intre telefon si subsisteme.

Subsisteme

Acces, Iluminat, Sanitar

CPU Principal si modulele conectate executa comenzile si publica starea reala a instalatiei.

Functionalitati

Acces

  • Control garaj: inchide, stop, deschide.
  • Control poarta: blocheaza si deblocheaza.
  • Autentificare biometrica pentru comenzi, cu exceptie in modul Debug.
  • Afisare camera de supraveghere si status Acces prin MQTT.

Iluminat

  • Control pentru 8 circuite de iluminat.
  • Actualizare stare din statusul MQTT de 8 biti.
  • Comenzi rapide din interfata Android.

Instalatie Sanitara

  • Afisare niveluri pentru doua recipiente.
  • Status valva, pompa si debit.
  • Mod AUTO/MANUAL si comenzi pentru valva si pompa.

Comunicatie

Comunicatia principala se face prin MQTT. Pentru camera si statusul de fallback al subsistemului Acces se foloseste HTTP.

Subsistem Topicuri principale Rol
Acces LICENTA/ACCES/GARAJ, LICENTA/ACCES/POARTA, LICENTA/ACCES/STATUS Comenzi garaj/poarta si status JSON pentru pagina Acces.
Iluminat LICENTA/ILUMINAT/COMANDA, LICENTA/ILUMINAT/STATUS Comenzi circuite si status pe 8 biti.
Instalatie Sanitara LICENTA/SANITAR/COMANDA, LICENTA/SANITAR/STATUS Comenzi mod/valva/pompa si status pentru niveluri, valva, pompa si debit.
