Schema hardware si alimentari

Pagina dedicata conexiunilor hardware pentru programare, alimentarii placii ESP12F_Relay_30A_X8 si recomandarilor de siguranta.

Atentie: nu face programarea sau probele initiale cu sarcini de 230V conectate la relee. Lucreaza intai doar cu alimentarea de control si conexiunile seriale.

1. Arhitectura de baza

Pentru scrierea firmware-ului, legatura recomandata este:

PC -> Arduino UNO (USB-Serial) -> ESP12F_Relay_30A_X8

  • PC-ul ruleaza Arduino IDE si trimite sketch-ul.
  • Arduino UNO este folosit doar ca adaptor USB-Serial.
  • Placa ESP12F_Relay_30A_X8 primeste firmware-ul pe UART.

2. Schema de conexiuni pentru programare

Arduino UNO ESP12F_Relay_30A_X8 Explicatie
GND GND Masa comuna intre adaptor si placa ESP
D0 / RX TX Receptie pe UNO din transmisia ESP
D1 / TX RX Transmitere din UNO spre receptia ESP
5V 5V Alimentare de control, doar daca placa este alimentata astfel in mod sigur
GND GPIO0 Forteaza intrarea in flash mode in timpul upload-ului
RESET -> GND - Dezactiveaza ATmega328P de pe UNO pentru a lasa doar USB-Serial-ul activ

3. Alimentarea placii

Alimentare pentru programare

  • Foloseste o alimentare stabila pentru placa de relee.
  • Mentine masa comuna intre UNO si placa ESP.
  • Daca alimentezi din pinul 5V al UNO, verifica atent ca placa sa accepte acest mod fara instabilitati.

Alimentare pentru exploatare

  • Placa este destinata functionarii cu alimentare separata, conform specificatiilor ei.
  • Evita sa bazezi functionarea normala a modulului pe alimentarea oferita de portul USB al UNO.
  • Conecteaza sarcinile de putere numai dupa ce firmware-ul este incarcat si testele de baza sunt finalizate.

4. Pasi recomandati pentru programare

  1. Conecteaza Arduino UNO la PC prin USB.
  2. Leaga RESET de pe UNO la GND.
  3. Realizeaza conexiunile UART si masa comuna.
  4. Leaga GPIO0 de pe placa ESP la GND.
  5. Alimenteaza placa ESP.
  6. Ruleaza upload-ul din Arduino IDE.
  7. Dupa finalizare, scoate GPIO0 din GND si reporneste placa.

5. Observatii de siguranta

  • Nu manipula sarcini de 230V in timpul programarii.
  • Nu lasa conexiuni improvizate pe alimentarea de putere.
  • Daca apar resetari sau erori seriale, verifica prima data alimentarea si masa comuna.
  • Daca upload-ul esueaza repetat, reduce viteza de upload si reporneste placa cand apare Connecting....
