Proiect Licenta - Modul Iluminat

Pagina de prezentare pentru subsistemul de iluminat bazat pe placa ESP12F_Relay_30A_X8, integrat in reteaua CPU Principal si controlat prin MQTT.

Rolul modulului

Modulul de iluminat controleaza 8 relee si ramane in subreteaua 192.168.4.0/24 a CPU Principal. El foloseste:

  • IP fix: 192.168.4.2
  • gateway: 192.168.4.1
  • DNS: 192.168.4.1
  • broker MQTT: 192.168.4.1:1883

Topicurile folosite raman:

  • LICENTA/ILUMINAT/COMANDA
  • LICENTA/ILUMINAT/STATUS

Ce gasesti in documentatie

  • pasii de conectare PC -> Arduino UNO -> ESP12F_Relay_30A_X8
  • setarile necesare in Arduino IDE
  • configurarea firmware-ului pentru WiFi, IP static si MQTT
  • formatul mesajelor MQTT si pasii de upload
