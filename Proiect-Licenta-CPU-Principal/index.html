Proiect Licenta

CPU Principal

Nucleul de integrare al proiectului ruleaza pe Raspberry Pi 4 si unifica retea locala, broker MQTT, orchestrare de stare si interfata tactila locala pe ecranul rezistiv de 3.5".

Instructiuni de instalare Hardware Software Subsisteme Structura repo
Raspberry Pi 4 CPU principal al sistemului
480x320 Rezolutia ecranului tactil local
MQTT + HTTP Canale de integrare intre module
AP local Retea Wi-Fi creata de Raspberry Pi

Platforma hardware

Proiectul foloseste un Raspberry Pi 4 Model B montat intr-o carcasa cu ecran touchscreen rezistiv de 3.5 inch, rezolutie 480x320, radiator si ventilator.

Broadcom BCM2711 Quad-core Cortex-A72 Wi-Fi 2.4/5 GHz GPIO 40 pini

De ce aceasta alegere

  • Ruleaza aplicatia principala si serviciile de integrare.
  • Poate afisa direct interfata locala pentru operator.
  • Poate gazdui retea Wi-Fi locala in modul Access Point.
  • Poate rula broker MQTT si servicii systemd la boot.

Arhitectura software

Retea locala

Raspberry Pi porneste in modul AP si creeaza reteaua la care se conecteaza subsistemele. 

SSID: Licenta-RPi-AP
IP AP: 10.42.0.1

Broker MQTT

Mosquitto ruleaza local si asigura schimbul de mesaje intre CPU Principal si module. 

Broker: 10.42.0.1
Port: 1883

UI local

Aplicatia din ecran/ ruleaza pe display-ul tactil si expune controale pentru operator. 

Tabs: Iluminat
      Acces
      Sanitar

Structura repository-ului

Documentatie

  • readme.md - prezentare generala
  • tutorial-instalare-rpi4-ap.md - Linux + Access Point
  • tutorial-instalare-mqtt-broker.md - broker MQTT

Componente executabile

  • scripts/ - serviciu de orchestrare MQTT + autostart
