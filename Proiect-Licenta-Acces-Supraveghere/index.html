Sistem automatizare instalatii electrice tip Smart-House folosind Arduino

Proiect Licenta Acces Supraveghere

Sistem IoT pentru control acces si supraveghere, cu Raspberry Pi ca infrastructura Wi-Fi/MQTT, camera OV5640, control garaj cu stepper prin TMC2209 si blocare poarta cu micro servo.

Deschide instructiunile Schema hardware

Scopul proiectului

Proiectul unifica accesul fizic si supravegherea video intr-un firmware pentru placi ESP32. Comenzile vin prin MQTT de pe Raspberry Pi, iar utilizatorul poate controla garajul si poarta dintr-o interfata web locala.

Documentatia pas cu pas este in docs/instructiuni.html.

Functionalitati

  • Conectare la reteaua Wi-Fi creata de Raspberry Pi.
  • Comenzi MQTT pentru garaj si poarta.
  • Status JSON retained pe LICENTA/ACCES/STATUS.
  • Interfata web pentru control local.
  • Camera OV5640 pentru supraveghere.
  • Configuratie pe card microSD / TF.

Componente

  • ESP32-S3-N16R8 ESP-CAM sau FireBeetle 2 ESP32-P4 + IO Expansion
  • Camera OV5640
  • Driver stepper TMC2209
  • Motor Bondtech NEMA17 Pancake Stepper 22mm
  • Micro servo 9g
  • Raspberry Pi pentru Wi-Fi si MQTT

Topicuri MQTT

LICENTA/ACCES/GARAJ

Payload-uri: deschide, stop, inchide.

LICENTA/ACCES/POARTA

Payload-uri: blocheaza, deblocheaza.

LICENTA/ACCES/STATUS

Status JSON retained publicat la conectare, dupa comenzi si la finalul miscarii garajului.
