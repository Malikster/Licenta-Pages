Sistem automatizare instalatii electrice tip Smart-House folosind Arduino

Instructiuni ESP32-S3-N16R8 IoT ESP-CAM Dev Board

Ghid dedicat variantei Arduino pentru placa ESP32-S3-N16R8 ESP-CAM cu camera OV5640, microSD onboard, TMC2209 si micro servo.

Alege alta placa Schema hardware

1. Componente folosite

  • ESP32-S3-N16R8 IoT ESP-CAM Dev Board cu camera OV5640
  • driver stepper TMC2209
  • motor Bondtech NEMA17 Pancake Stepper 22mm
  • micro servo 9g
  • card microSD
  • sursa principala 12V si buck 9-35V -> 5V 5A

2. Alimentare

Sursa 12V
  +--> VMOT TMC2209
  +--> intrare buck 9-35V -> 5V

Buck 5V
  +--> 5V ESP32-S3
  +--> 5V servo

GND comun:
  sursa 12V
  buck
  ESP32-S3
  TMC2209
  servo
Nu alimenta stepper-ul din 5V si nu alimenta servo-ul din 3.3V.

3. Wiring ESP32-S3

FunctiePin ESP32-S3
TMC2209 STEPGPIO1
TMC2209 DIRGPIO14
TMC2209 EN/ENNGPIO21
Servo semnalGPIO42
microSD onboardCLK=39, CMD=38, D0=40

Camera OV5640 este definita in sketch si folosita automat pe acest profil.

4. Pregatirea cardului microSD

  1. Formateaza cardul in FAT32.
  2. Copiaza in radacina cardului fisierele din licenta-acces-supraveghere/sdcard.
  3. Redenumeste sau creeaza acces.conf pornind de la acces.conf.example.
WIFI_SSID=Licenta-RPi-AP
WIFI_PASSWORD=licenta-rpi
MQTT_HOST=192.168.4.1
MQTT_PORT=1883
MQTT_USERNAME=licenta
MQTT_PASSWORD=licenta
MQTT_CLIENT_ID=licenta-acces-esp32s3
STATIC_IP=192.168.4.3
GARAGE_TRAVEL_STEPS=2400
GARAGE_STEP_INTERVAL_US=2500
TMC_ENABLE_AFTER_MOVE=false
TMC_INVERT_DIRECTION=false
SERVO_LOCKED_DEG=20
SERVO_UNLOCKED_DEG=95

5. Arduino IDE

  1. Instaleaza Arduino IDE 2.x.
  2. In Boards Manager, instaleaza esp32 by Espressif Systems.
  3. In Library Manager, instaleaza PubSubClient.
  4. Deschide licenta-acces-supraveghere/licenta-acces-supraveghere.ino.

Setari recomandate

  • Board -> ESP32S3 Dev Module
  • USB CDC On Boot -> Enabled
  • Flash Size -> 16MB
  • PSRAM -> OPI PSRAM sau Enabled
  • Upload Speed -> 460800 sau 921600

6. Upload si pornire

  1. Apasa Verify in Arduino IDE.
  2. Daca build-ul trece, apasa Upload.
  3. Daca upload-ul nu porneste, tine apasat BOOT, apasa scurt RST/EN, apoi elibereaza BOOT.
  4. Deschide Serial Monitor la 115200 baud.

7. Testare web

  • http://IP_PLACA/ pentru pagina de control
  • http://IP_PLACA/status pentru status JSON
  • http://IP_PLACA/camera.jpg pentru captura camera
  • http://IP_PLACA/upload.html pentru administrare SD

8. Test MQTT

mosquitto_pub -h 192.168.4.1 -u licenta -P licenta -t LICENTA/ACCES/GARAJ -m deschide
mosquitto_pub -h 192.168.4.1 -u licenta -P licenta -t LICENTA/ACCES/GARAJ -m stop
mosquitto_pub -h 192.168.4.1 -u licenta -P licenta -t LICENTA/ACCES/GARAJ -m inchide
mosquitto_pub -h 192.168.4.1 -u licenta -P licenta -t LICENTA/ACCES/POARTA -m deblocheaza
mosquitto_pub -h 192.168.4.1 -u licenta -P licenta -t LICENTA/ACCES/POARTA -m blocheaza
mosquitto_sub -h 192.168.4.1 -u licenta -P licenta -t LICENTA/ACCES/STATUS -v

LICENTA/ACCES/STATUS este publicat retained la conectare, dupa comenzi si la finalul miscarii garajului.

9. Troubleshooting

  • Daca stepper-ul vibreaza, verifica bobinele A1/A2 si B1/B2.
  • Daca motorul merge invers, seteaza TMC_INVERT_DIRECTION=true.
  • Daca servo-ul reseteaza placa, imbunatateste alimentarea de 5V.
  • Daca pagina web lipseste, verifica fisierele de pe microSD.
