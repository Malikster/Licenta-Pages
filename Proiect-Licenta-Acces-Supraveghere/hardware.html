Sistem automatizare instalatii electrice tip Smart-House folosind Arduino

Schema hardware si alimentari

Conexiuni recomandate pentru ESP32-S3 sau FireBeetle 2 ESP32-P4, driver TMC2209, motor Bondtech NEMA17 Pancake 22mm, micro servo, camera OV5640 si alimentarea comuna.

Instructiuni complete Prezentare proiect

Schema generala de alimentare

Sursa principala 12V
  +--> VMOT TMC2209
  +--> intrare buck 9-35V -> 5V

Buck 5V
  +--> 5V / VIN placa ESP32
  +--> +5V servo

ESP32 3.3V
  +--> VIO / VDD logic TMC2209, daca modulul are acest pin

GND comun
  sursa 12V
  buck
  ESP32
  TMC2209
  servo
Nu alimenta stepper-ul din 5V, nu alimenta servo-ul din 3.3V si nu conecta VMOT direct la placa ESP32.

Recomandari electrice

  • Regleaza buck-ul la 5V inainte sa conectezi ESP32 si servo.
  • Leaga toate masele impreuna: sursa, buck, ESP32, TMC2209 si servo.
  • Pune un condensator 100uF - 470uF aproape de VMOT/GND pe TMC2209.
  • Pune un condensator 470uF - 1000uF pe linia de 5V daca servo-ul reseteaza placa.
  • Foloseste radiator pe TMC2209 si verifica temperatura dupa primele teste.

Ordine de testare

  1. Testeaza ESP32 singur, alimentat din USB sau 5V stabil.
  2. Testeaza servo-ul fara mecanica montata.
  3. Testeaza TMC2209 fara motor sau cu motorul liber, fara sarcina.
  4. Monteaza mecanica doar dupa ce directia si capetele de cursa sunt clare.
Pentru primele teste la motorul Bondtech NEMA17 Pancake 22mm, porneste conservator cu aproximativ 0.6A - 0.8A RMS pe TMC2209.

Conexiuni comune TMC2209

Pin / functie TMC2209 Conectare Observatii
VMOT +12V din sursa motorului Nu se leaga la 5V ESP32.
GND motor GND sursa 12V si GND comun Necesar pentru referinta comuna.
VIO / VDD, daca exista 3.3V ESP32 Alimenteaza logica driverului la nivel compatibil cu ESP32.
STEP, DIR, EN/ENN Pinii ESP32 din tabelul placii folosite EN/ENN este activ LOW pe majoritatea modulelor TMC2209.
A1/A2 si B1/B2 Cele doua bobine ale motorului stepper Identifica perechile de bobine cu multimetrul inainte de alimentare.
Nu conecta sau deconecta motorul stepper cat timp TMC2209 este alimentat.

Checklist inainte de alimentare

  • Buck-ul este masurat la 5V cu multimetrul.
  • GND este comun intre toate modulele.
  • TMC2209 are radiator si curent setat conservator.
  • Bobinele motorului sunt identificate corect.
  • Servo-ul este alimentat din 5V, nu din 3.3V.
  • Pinii P4 rezervati pentru camera, TF card si ESP32-C6 nu sunt folositi pentru actuatori.
