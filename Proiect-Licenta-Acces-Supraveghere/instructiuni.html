ESP32-S3-N16R8 IoT ESP-CAM Dev Board

Foloseste sketch-ul Arduino principal si camera OV5640 din profilul ESP32-S3.

  • Arduino IDE / Arduino CLI
  • camera JPEG prin /camera.jpg
  • interfata web de pe microSD
  • MQTT pentru garaj, poarta si status
Instructiuni ESP32-S3

FireBeetle 2 ESP32-P4 + IO Expansion Board

Foloseste subproiectul ESP-IDF pentru camera OV5640 pe MIPI-CSI si control acces in acelasi firmware.

  • ESP-IDF
  • stream MJPEG pe http://IP:81/stream
  • pagina de control la /acces
  • MQTT pentru garaj, poarta si status
Instructiuni ESP32-P4

Topicuri comune MQTT

  • LICENTA/ACCES/GARAJ: deschide, stop, inchide
  • LICENTA/ACCES/POARTA: blocheaza, deblocheaza
  • LICENTA/ACCES/STATUS: status JSON retained
Raspberry Pi ramane host pentru Wi-Fi si broker MQTT in ambele variante.
